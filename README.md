# Real-Time Phishing URL Detection System

A Machine Learning and Flask-based web application that detects phishing URLs using feature extraction and threat analysis.

## Features

- Machine Learning phishing detection
- Flask web application
- URL feature extraction
- Confidence scoring
- Real-time threat analysis
- Clean UI dashboard

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- HTML/CSS

```bash
python train_model.py
py app.py
